How to use this project?
Follow this steps

step 1--Open your terminal and git clone https://github.com/Nihal-ls/share-circle.git

step 2-- npm i
step-3 npm run dev 


Follow this steps to connect in the git
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Nihal-ls/share-circle.git
git push -u origin main
